# 小红书/公众号爆款逻辑

## 小红书爆款逻辑

### 1. 黄金3秒
- 前3秒必须抓住注意力
- 方法：痛点前置、悬念开场、数字冲击

### 2. 内容结构
```
痛点共鸣 → 解决方案 → 效果展示 → 行动号召
```

### 3. 爆款标题公式
- 数字法：「3分钟学会XXX」
- 对比法：「从0到1，我用XXX彻底改变了XXX」
- 悬念法：「别找了，XXX就那么简单」
- 痛点法：「卷死同事！快用XXX恢复你的职场执行力」

### 4. 视觉节奏
- 5秒内入题
- 每10秒画面变化
- 信息密度高但节奏快

### 5. 互动设计
- 结尾提问："你还知道什么隐藏玩法吗？"
- 引导评论："评论区聊聊"
- 号召行动："点我头像发就好啦"

### 6. 关键词
- 小白入门
- 彻底改变
- 低成本/免费/零积分
- 24小时在线
- 卷死同事

## 公众号爆款逻辑

### 1. 标题公式
```
数字 + 痛点 + 解决方案
```

**示例**：
- 「Obsidian+OpenClaw组合技！花9分钟重构你的AI知识管理体系」
- 「高级OpenClaw第一步！先用12个Skills把联网搜索做到极致」
- 「Nano Banana 2对决Pro纯享版，我跑完了20个真实案例和提示语！」

### 2. 文章结构

#### 开头（故事/场景代入）
- 个人经历："三周前我决定迁移到Claude Code+Obsidian..."
- 痛点描述："这其实挺难的..."
- 转折："但这有个很无解的缺点..."

#### 中间（干货+案例+截图）
- 分步骤讲解
- 大量截图佐证
- 对比展示（Before/After）
- 具体数据和案例

#### 结尾（金句+行动号召）
- 总结升华："我想这就是跟AI共用知识体系才会有的独特体验..."
- 行动号召："后台回复'龙虾'就行"
- 互动引导："如果你有更有趣的玩法，欢迎在评论区聊聊"

### 3. 排版技巧
- 小标题清晰
- 重点加粗
- 图片丰富（截图、对比图、效果图）
- 适当的emoji

### 4. 互动运营
- 评论区回复读者
- 引导私信获取资料
- 引导关注、点赞、转发

## 通用爆款原则

### 1. 价值感
- 看完能学到东西
- 有可执行的步骤
- 有具体的案例

### 2. 情绪价值
- 共鸣："你是不是也遇到过..."
- 惊喜："这波这波是赚大了"
- 成就感："搞定！收工！"

### 3. 传播价值
- 有金句可以转发
- 有截图可以分享
- 有资料可以索取

## 脚本反馈时的应用

分析脚本时，从以下角度评估爆款潜力：

1. **标题吸引力**（1-10分）
2. **开场钩子**（1-10分）
3. **内容价值**（1-10分）
4. **节奏感**（1-10分）
5. **互动设计**（1-10分）

总分50分，换算成10分制给出综合评分。
