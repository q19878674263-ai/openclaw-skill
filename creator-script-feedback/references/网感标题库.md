## 网感公式

**基础公式**：
人设/身份 + 解决方案 + 结果/价值 + （情绪/悬念/数字）

**进阶公式**：
- 痛点前置 + 解决方案 + 结果反差
- 身份反差 + 黑话 + 价值承诺
- 数字冲击 + 稀缺感 + 行动号召

## 标题拆解模板

分析一个网感标题时，拆解：
1. **钩子是什么**（前5个字抓住你）
2. **痛点是什么**（解决了什么问题）
3. **解决方案是什么**（怎么解决的）
4. **价值承诺是什么**（能得到什么）
5. **情绪触发是什么**（为什么想点）

## 创作原则（重要！）

### ❌ 不要做的
1. **不要套模板** - 每篇内容不同，标题必须结合具体卖点
2. **不要重复用词** - 不同视频标题要有差异化
3. **不要堆砌关键词** - 不是网感词越多越好

### ✅ 要做的
1. **先理解内容** - 这篇的核心卖点是什么？（记忆/沉淀/长期资产）
2. **找到差异化** - 和上一篇的区别是什么？（信息获取 vs 知识沉淀）
3. **突出化学反应** - 工具组合要有1+1>2的效果（OpenClaw+Obsidian=会记忆的AI）
4. **用新角度** - 金融化（长期资产）、拟人化（会记忆）、联名感（✖️）
