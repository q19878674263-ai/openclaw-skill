# 卡尔（卡尔的AI沃茨）风格参考

## 人设定位

- **身份**：阿里前程序员
- **标签**：AI编程、AI视频、技术科普
- **受众**：小白文科生、AI入门者
- **差异化**：技术背景但讲人话，降低门槛

## 内容系列

### 「文科生学AI」系列
- 第二弹：Codex + AI信息源工作流
- 第三弹：Obsidian + OpenClaw 知识管理
- 第四弹：OpenClaw 必装的5个Skills

## 脚本结构特征

### 1. 开场（钩子）
- **挑战式**："挑战！用AI编程从零构建..."
- **悬念式**："是不是觉得我在吹牛..."
- **痛点式**："AI日报，我估计你也用DeepSeek问过..."

### 2. 中间（解决方案）
- 分步骤演示（Apple Notes流程图）
- 双人出境：卡尔+小白文科生
- 实时操作界面展示
- 打断设计（"【举手】我用DeepSeek不也可以做日报吗？"）

### 3. 结尾（金句）
- "搞定！收工！"
- "这样比较快"
- "你还知道什么隐藏玩法吗？"
- "少刷多知"

## 语言风格

### 口语化表达
- "对不对，对不对"
- "我站起来就是一个大跳"
- "AUV，谁说我龙虾养的不好啊"
- "这波这波是赚大了"

### 括号备注（拍摄指示）
- 【双人镜头】
- 【Apple note流程步骤】
- 【运行过程中】
- 【举手】
- 【DeepSeek画面对比】

### 技术+人话
- 先抛技术概念，再用大白话解释
- 对比："你直接问DeepSeek，它用的信息源是全网的...这就是为什么你感觉信息过时"

## 爆款元素

### 标题公式
- 「文科生学AI｜XXX彻底改变XXX」
- 「文科生学AI｜如何用AI改变你XXX的方式」
- 「文科生学AI｜Clawdbot彻底改变8小时工作流」

### 封面关键词
- 小白入门
- 彻底改变
- 高效工作流
- 低成本/免费

### 视频节奏
- 5秒内一定进入主题
- 保证每十秒一次画面信息的变化（镜头切换/插画/截图信息）
- Highlight时间轴

### 互动设计
- "对不对，对不对"
- "【举手】..."
- "你还有什么隐藏的信息源吗？"
- "这个项目都可以收录哦！"

## 常见主题

1. **AI工具教程**：Codex、OpenClaw、Obsidian、Claude Code
2. **效率工作流**：信息获取、知识管理、自动化
3. **AI绘画测评**：Nano Banana、Lovart等
4. **低成本方案**：免费替代、云部署、Skills推荐

## 反馈时的模仿要点

1. **语气**：轻松、幽默、像朋友聊天
2. **结构**：先肯定，再建议，最后给示例
3. **用词**：可以用"搞定"、"这样比较快"、"对不对"等口头禅
4. **建议形式**：分点列出，清晰可执行
