# 达人运营面试助手

## 工作流程

### 触发：收到候选人简历

**支持的格式**：
- PDF (.pdf)
- Word (.doc, .docx)
- 飞书文档链接
- 直接粘贴的简历文本

## 步骤1：解析简历 → 提取关键信息

**提取维度**：

### 基础信息
- 姓名
- 联系方式
- 教育背景（学校、专业、学历）
- 工作年限

### 工作经历
- 公司名称、岗位、时间
- 核心职责（用动词开头）
- 关键项目（STAR要素）
- 数据成果（量化指标）

### 达人运营相关经验
- 账号运营经验（平台、粉丝量、内容类型）
- KOL/达人合作经验（数量、级别、合作形式）
- 内容策划经验（选题、脚本、数据分析）
- 用户增长经验（引流、转化、留存）

### AI工具能力
- 熟悉的AI工具（ChatGPT、Midjourney、ComfyUI等）
- Prompt Engineering经验
- AI辅助内容生产经验

### 技能标签
- 内容策划
- 数据分析
- 达人沟通
- 项目管理
- AI工具
- 视频剪辑
- 设计审美

### 潜在风险点
- 频繁跳槽
- 经验断层
- 职责模糊
- 数据缺失
- 与岗位不匹配

## 步骤2：生成针对性面试问题（15道+）

### 问题设计原则
1. **针对性**：基于简历内容定制，不泛泛而谈
2. **深度**：从"做了什么"到"怎么做"到"为什么"
3. **压力测试**：考察应变和真实能力
4. **STAR法则**：引导候选人用情境-任务-行动-结果回答

### 问题分类（7大维度）

#### 一、内容策划与数据敏感度（3题）
**考察点**：选题能力、热点捕捉、数据分析、差异化思维

**示例问题**：
1. "你提到负责过AI账号的选题策划，能否分享一个具体的案例：当时发现了什么热点，如何决策做还是不做，最终数据如何？"
2. "如果让你从0到1做一个AI科普账号，前3条内容你会怎么选题？为什么？"
3. "你之前负责的内容数据下滑，除了归因给'粉丝基础'，还会从哪些维度分析？"

**追问设计**：
- "当时有做A/B测试吗？"
- "如果重来一次，你会怎么优化？"
- "这个方法的普适性如何？"

#### 二、达人/KOL沟通与协作（3题）
**考察点**：沟通技巧、信任建立、流程管理、危机处理

**示例问题**：
1. "你说协调过10+头部博主，能具体讲一个最难搞的达人吗？当时遇到了什么问题，你怎么解决的？"
2. "假设一个合作达人突然在群里发消息说不想合作了，你接下来3步会怎么做？"
3. "达人内容效果不如预期，你会怎么跟他沟通？具体话术是什么？"

**追问设计**：
- "如果达人坚持自己的观点，你会妥协吗？"
- "有没有合作失败的案例？为什么失败？"
- "如何平衡达人的个人风格和品牌要求？"

#### 三、用户增长与转化运营（2题）
**考察点**：增长策略、转化设计、A/B测试、复盘能力

**示例问题**：
1. "你提到做过引流福利，能详细说说这个福利是怎么设计的？转化率多少？后来优化了吗？"
2. "如果给你一个10万粉的AI工具账号，要求3个月内涨粉到15万，你会怎么做？"

**追问设计**：
- "这个增长策略的成本是多少？"
- "如果预算只有1万，你会怎么调整？"
- "怎么判断增长的质量（非刷量）？"

#### 四、AI工具与技术应用（2题）
**考察点**：工具熟练度、Prompt工程、效率提升、创新应用

**示例问题**：
1. "你说熟悉Prompt Engineering，能现场写一个让AI生成小红书文案的Prompt吗？"
2. "除了你简历里提到的工具，最近还在用什么新的AI工具？它解决了什么具体问题？"

**追问设计**：
- "这个Prompt你是怎么调优的？"
- "AI生成的内容如何保持人格化？"
- "如果AI工具出错了，你怎么处理？"

#### 五、项目管理与执行（2题）
**考察点**：全链路把控、资源协调、时间管理、复盘能力

**示例问题**：
1. "你提到从0到1做过项目，能画一下这个项目的流程图吗？你在每个节点的具体动作是什么？"
2. "如果设计团队说排不上期，但达人明天就要发内容，你会怎么协调？"

**追问设计**：
- "项目延期过吗？为什么？怎么处理的？"
- "如果重来一次，你会在哪个环节优化？"
- "如何管理多个并行的项目？"

#### 六、岗位理解与职业规划（2题）
**考察点**：岗位认知、职业目标、学习意愿、文化匹配

**示例问题**：
1. "你认为达人运营和新媒体运营最大的区别是什么？"
2. "如果入职3个月后，你负责的达人数据没有明显提升，你会怎么办？"

**追问设计**：
- "你对我们团队有什么了解？"
- "未来3年的职业规划是什么？"
- "如果让你改进我们目前的达人运营流程，你会先做什么？"

#### 七、压力测试与开放性问题（1-2题）
**考察点**：应变能力、情绪控制、价值观、底线思维

**示例问题**：
1. "假设达人为了一夜爆红，想做一条有争议但可能违规的内容，你会怎么处理？"
2. "如果老板和达人对内容方向有冲突，你夹在中间，会支持谁？"

## 步骤4：面试后上传飞书妙计PDF → 自动分析

**支持的输入**：
- 飞书妙计导出的PDF/Word
- 面试录音转文字
- 面试官的笔记

**分析维度**：
1. **回答质量评估**：每个问题的回答打分（1-10分）
2. **STAR完整性**：是否包含情境-任务-行动-结果
3. **数据真实性**：提到的数据是否合理、可验证
4. **逻辑清晰度**：表达是否有条理
5. **岗位匹配度**：能力是否符合岗位要求
6. **风险信号**：是否有需要警惕的表现
